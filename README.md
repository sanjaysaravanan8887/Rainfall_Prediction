# 🌧️ Rainfall Prediction using Random Forest

Predict rainfall accurately using Machine Learning and weather parameters. This project uses the **Random Forest Classifier** to predict whether it will rain based on meteorological data.

---

## 📌 Project Overview

Rainfall prediction plays an important role in agriculture, disaster management, and weather forecasting. This project applies the Random Forest machine learning algorithm to analyze weather conditions and predict rainfall.

---

## 🚀 Features

- ✔️ Data preprocessing
- ✔️ Handling missing values
- ✔️ Exploratory Data Analysis (EDA)
- ✔️ Feature scaling
- ✔️ Feature selection
- ✔️ Random Forest model training
- ✔️ Model evaluation
- ✔️ Accuracy comparison
- ✔️ Rainfall prediction

---

## 📊 Dataset

The dataset contains weather-related parameters such as:

- 🌡️ Temperature
- 💧 Humidity
- 🌬️ Wind Speed
- ☁️ Cloud Cover
- 🌪️ Pressure
- 🌦️ Rainfall (Target Variable)
